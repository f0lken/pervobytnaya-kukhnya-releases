# Pervobytnaya Kukhnya Releases

Public Android release builds for `Первобытная кухня`.

## What Is Here

This repository is used only for published Android release artifacts.

- Latest APK builds are published on the [Releases](https://github.com/f0lken/pervobytnaya-kukhnya-releases/releases) page
- The main source repository remains private

## Current Release

- Version: `1.0.0+1`
- Platform: Android APK
- Package: `com.example.pervobytnaya_kukhnya`

## Install

1. Open the latest release on the [Releases](https://github.com/f0lken/pervobytnaya-kukhnya-releases/releases) page
2. Download the APK asset
3. Install it on your Android device

## Integrity Check

Each release includes a `.sha256.txt` file next to the APK so you can verify the download locally.
